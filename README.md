GooglePhotosTrandistion: 一个模仿GooglePhoto之前所拥有的圆形展开以及消失的动画（目前手势操作动画会有bug。根据Rob Napier和Mughunth Kumar在《IOS编程实战》中所说明，苹果的自定义转场手势不支持layer动画。所以得自己计算百分比，并且布置mask的面积）。
